# مُبلِّغ 🚨

شبكة الحماية المجتمعية من الاحتيال المالي الرقمي — جزء من مشروع الحماية الرقمية.

## رفع الموقع على GitHub Pages

1. أنشئ repository جديد على GitHub
2. ارفع محتوى هذا المجلد كاملاً (مع مجلدات css/ و js/ و blog/)
3. Settings → Pages → main branch → Save
4. موقعك: `https://username.github.io/repo-name`

## الصفحات

- `index.html` — الرئيسية
- `feed.html` — البلاغات الحية مع فلتر
- `report.html` — نموذج الإبلاغ
- `dictionary.html` — قاموس الفخاخ (12 مصطلح)
- `about.html` — من نحن
- `blog/` — 6 مقالات
